# 城市调查报告标准模板(10 节结构)

> 用法:执行完 11 次 API 调用后,按本模板整合 Markdown 报告。
> 文件名约定:`{城市}{时间}调查报告_博查版.md`,例如 `广州下周末调查报告_博查版.md`。

---

## 报告头(必填)

```markdown
# {城市}{时间}调查报告 · 博查版

> 调查日期:{YYYY-MM-DD}
> 覆盖时段:{本周末/下周末/未来一个月}+ 月度排期 + 暑期长效活动
> 调查方式:博查 WebSearch API
> API 调用次数:**N 次**(预算 50,余量 X%)
> 信源:{列出主要权威媒体,如腾讯新闻、网易、人民网、官方文旅、本地宝等} 等 N+ 家

---
```

## 图文报告要点

### 图片选择策略

- 每节选 **1-2 张代表图**,不要堆砌
- 优先选 `siteName` 是权威媒体的(腾讯/官方)
- 避开明显是广告或低质的图(通过 thumbnailUrl 域名判断)
- 检测 `//` 开头要补 `https:`

### 图片嵌入格式

```markdown
![{描述}]({url})

**图:{图片说明}(来源:{siteName} {date})**
```

**关键**:每张图下面**必带一行粗体说明**,即使图失效,读者也能从文字获取信息。

### 图片提取 python 脚本

```python
import json

def extract_images(json_files):
    """从博查响应中提取可用图片 URL"""
    images = []
    for f in json_files:
        with open(f, encoding='utf-8') as fp:
            d = json.load(fp)
        imgs = (d.get('data',{}).get('images') or {}).get('value',[])
        for img in imgs:
            url = img.get('thumbnailUrl') or img.get('contentUrl') or ''
            if url.startswith('//'):
                url = 'https:' + url
            if url.startswith('http') and 'icon' not in url.lower() and 'logo' not in url.lower():
                images.append((f, url, img.get('name','')[:50]))
    return images
```
